# 气质方向库（Aesthetic Directions）

定调时从这里选一个**极端**方向，或在此基础上自创。
原则：**宁可走极端走出记忆点，也不要选"现代简约"这种安全的废话。**

每个方向给出可直接落地的打法，避免"选了方向却不知道怎么做"。

---

## 1. 凶悍 / 工业（Brutalist）

气质：硬核、高对比、不讨好、有攻击性。适合开发者工具、硬科技、有态度的品牌。

- **色彩**：近黑 + 一个高饱和电光色（酸绿 #c8ff00 / 电蓝 / 警示橙）。绝不用柔和渐变。
- **字体**：超粗无衬线（Helvetica 加粗、Archivo、Space Grotesk）+ 等宽体做点缀。字号对比拉到极致。
- **排版**：粗黑边框、硬阴影（offset 不模糊）、网格裸露、左对齐为主、刻意的不对称。
- **动效**：硬切、瞬时反馈，少缓动；或干脆无动效，靠静态张力。
- **签名时刻**：超大字号宣言占满一屏、裸露的网格线、故意的"未完成感"。

## 2. 优雅 / 高定（Editorial）

气质：杂志感、精致、有文化气、慢。适合内容品牌、设计工作室、高端产品、作品集。

- **色彩**：纸感米白/暖灰 + 一个克制的深色 + 至多一个点缀色。低饱和、有质感。
- **字体**：高质量衬线 display（GT Sermon、Playfair、Tiempos、思源宋体）+ 干净无衬线正文。字体本身就是设计。
- **排版**：杂志栏式网格、首字下沉、超大行距、文字与图错落、大量负空间。
- **动效**：缓慢、柔和的淡入与视差，克制优雅。
- **签名时刻**：一段超大号衬线引言、文字绕图排版、一张满血出血大图。

## 3. 克制 / 极简（Swiss）

气质：纪律、秩序、信息清晰、专业。适合 B2B、金融、数据产品、企业官网。

- **色彩**：黑白灰 + 一个功能性强调色。色彩极少，靠排版说话。
- **字体**：Neue Haas / Inter / Helvetica，严格的字号阶梯。
- **排版**：严格栅格、对齐到像素、大留白、信息层级用尺寸和间距而非颜色区分。
- **动效**：极少，仅功能性反馈。
- **签名时刻**：一个极致干净的数据可视化、一处打破栅格的精确留白。

## 4. 张扬 / 极繁（Maximalist）

气质：信息爆炸、撞色、热闹、年轻。适合活动页、潮流品牌、社区产品、Z 世代向。

- **色彩**：多色撞色、高饱和、渐变可以大胆用（但别用紫蓝那套烂大街的）。
- **字体**：多字体混搭、超大字、变形字、贴纸感。
- **排版**：密集、堆叠、重叠、旋转、贴纸/便签隐喻，故意打破栅格。
- **动效**：丰富、跳动、滚动触发的多重动画。
- **签名时刻**：一面"信息墙"、滚动时元素爆裂出现、夸张的鼠标跟随交互。

## 5. 复古 / 未来（Retro-Futuristic）

气质：赛博、霓虹、像素、Y2K、合成波。适合游戏、AI 产品、潮玩、音乐。

- **色彩**：深底 + 霓虹（品红/青/紫）+ 扫描线/噪点质感。或 Y2K 的金属银+电光。
- **字体**：像素字、等宽、科技感无衬线、故障字效。
- **排版**：终端窗口隐喻、网格地平线、HUD 元素、发光描边。
- **动效**：故障(glitch)、霓虹呼吸、打字机、扫描线。
- **签名时刻**：一个会发光/故障的标题、一片 3D 网格地平线、终端打字效果。

## 6. 有机 / 自然（Organic）

气质：柔和、温暖、手作感、亲切。适合健康、食品、母婴、可持续品牌。

- **色彩**：大地色、莫兰迪、低饱和暖色，柔和过渡。
- **字体**：圆润无衬线、手写体点缀、温暖的衬线。
- **排版**：圆角、有机形状（blob）、不规则边缘、手绘插画、纸张/布料质感。
- **动效**：柔软的缓动、轻微的漂浮感。
- **签名时刻**：手绘插画、有机形状的图片裁切、一处温暖的微交互。

## 7. 玩味 / 玩具（Playful）

气质：明亮、圆润、趣味、有点傻气的可爱。适合 C 端工具、教育、儿童、轻量 SaaS。

- **色彩**：明亮糖果色、高明度、对比愉悦。
- **字体**：圆体、胖胖的无衬线、有性格的字。
- **排版**：大圆角、立体卡片、拟物化小元素、夸张的图标。
- **动效**：弹跳(spring)、回弹、俏皮的 hover 反馈、拟人化微交互。
- **签名时刻**：一个会做表情/反应的吉祥物、夸张的弹跳交互、立体感的卡片堆叠。

## 跨方向通用的"反平庸"硬规则

无论选哪个方向，这几条都适用：

1. **字号对比至少拉到 1:4**。标题和正文不能差不多大。最大的那个字要敢于大到"有点过分"。
2. **强制选一个非默认字体**。系统默认字体（Arial / 苹方直接上）= 没做设计。
3. **色彩克制**：主色 + 至多两个辅助色 + 中性色。颜色多 ≠ 丰富，节奏才是丰富。
4. **留白服务节奏**：留白出现在模块之间制造呼吸，不是均匀撒在每个角落。
5. **至少一个大胆决策**：超大字、出血图、打破栅格、强动效……一个页面没有任何一个"敢"的决定，就是平庸。
