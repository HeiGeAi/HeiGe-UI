# 反 AI 体检清单（Anti-Slop Checklist）

出货前**强制**逐条过。命中任何一条 AI slop 特征，回去改。
这不是建议，是交付门槛。

---

## 一、致命 AI slop 特征（命中=必改）

这些是一眼就能看出"AI 做的"的特征，出现一个就回去重做那部分：

- [ ] **紫蓝渐变背景**：`#667eea → #764ba2` 及其全家桶。这是 AI 界面的身份证，绝对禁用。
- [ ] **居中卡片三件套**：一行标题 + 一行灰色副标题 + 三个等宽圆角卡片，整页重复这个结构。
- [ ] **系统默认字体直接上**：没有任何字体性格设计，Arial / Helvetica / 苹方裸奔。
- [ ] **千篇一律的圆角阴影**：每个元素都是 `border-radius: 8px` + `box-shadow: 0 4px 6px rgba(0,0,0,.1)`。
- [ ] **emoji 当图标**：用 emoji 撑场面、当功能图标。
- [ ] **通用 SaaS 模板感**：Hero + Features(3栏) + Testimonials + Pricing + Footer，每段都一个调子，毫无节奏。
- [ ] **正确的废话文案**：「赋能」「一站式」「智能高效」「让 X 更简单」这种填充词。

## 二、节奏体检（气场来自节奏）

- [ ] **首屏 3 秒能立住气质吗？** 截图给没看过的人，他能说出调性，才算过。
- [ ] **有张弛吗？** 是不是从头到尾一个密度、一个调子？强模块后面有没有"呼吸"？
- [ ] **高潮在哪？** 全页最想让人记住的那一屏，是不是被放在了节奏的高点上？
- [ ] **收束干脆吗？** 结尾有没有明确的行动号召或余味，还是拖泥带水戛然而止？

## 三、主角体检（一屏一主角）

- [ ] **每屏主角清晰吗？** 逐屏问：这屏的视觉主角是谁？说不出唯一答案，就是没主角。
- [ ] **配角压下去了吗？** 是不是每个元素都在抢戏、视觉权重都差不多？
- [ ] **对比够不够狠？** 主角和配角的尺寸/色彩/留白对比，敢不敢再拉大一倍？

## 四、签名时刻体检

- [ ] **有没有签名时刻？** 这一页有哪一个细节，是换个 AI、换个模板绝对做不出来的？
- [ ] 答得上来 → 过。答不上来 → 回去设计一个，别交。

## 五、字体与排版

- [ ] **选了非默认字体吗？** display / 正文 / mono 至少有一个有态度的选择。
- [ ] **字号对比够吗？** 最大字和正文至少 1:4，最大那个字敢不敢"大到有点过分"。
- [ ] **对齐有纪律吗？** 是不是无脑全居中？左对齐/网格对齐往往更有力量。
- [ ] **留白服务节奏吗？** 留白是在模块间制造呼吸，不是均匀撒满每个缝隙。

## 六、工程质量

- [ ] **代码是真能跑的吗？** 不是占位、不是 lorem ipsum 凑数。
- [ ] **响应式真做了吗？** 移动端是重新编排的另一段叙事，不是桌面端缩一缩。
- [ ] **细节经得起放大吗？** hover 状态、焦点态、加载态、空状态、边界情况有没有顺手做掉。
- [ ] **性能 OK 吗？** 图片有没有压、动画卡不卡、首屏快不快。
