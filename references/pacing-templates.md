# 节奏曲线模板（Pacing Templates）

节奏 = 气场的来源。不同页面类型有不同的张弛排布。
下面是常见类型的节奏参考。**模板是起点，不是终点**，拿到具体项目要按内容重新导演。

记号：⚡ 冲击 / 🫁 呼吸 / 📈 推进 / 🔥 高潮 / 🎬 收束

---

## 落地页 / 产品官网（Landing Page）

```
⚡ 开场定调    Hero：一句话立人设 + 主视觉/大字，3 秒定调
🫁 呼吸        一行价值主张 + 留白，让用户喘口气、消化
📈 推进①      核心能力/卖点，但要有主次，不是三个等宽卡片平铺
🫁 呼吸        一段引用/数据/留白过渡
📈 推进②      场景化展示 / 真实截图 / 工作流，越具体越可信
🔥 高潮        签名时刻放这里：最炸的一屏，最想被记住的瞬间
🎬 收束        强 CTA，一个动作，干脆利落
```

要点：高潮不一定在最后，可以是中后段的某一屏；CTA 收尾要短。

---

## Dashboard / 数据产品

```
⚡ 开场        最重要的那个核心指标，大到占据视觉中心（一屏一主角）
🫁 呼吸        次级指标用克制的方式排布，主动后退
📈 推进        主图表/主视图，信息密度可以高，但层级要清晰
🫁 呼吸        筛选/操作区，功能性，不抢戏
🔥 签名        一个做到极致干净的可视化，或一处巧妙的信息密度处理
```

要点：Dashboard 的气场来自"克制"和"信息层级"，不是花哨。主角是数据，不是装饰。

---

## 作品集 / Portfolio

```
⚡ 开场        强烈的个人气质表达，一句话/一个动作立住"你是谁"
🫁 呼吸        简短自我介绍，留白要足
📈 推进        作品逐个展示，每个作品自己就是一屏一主角
🔥 高潮        最强的那个作品 / 一个签名级的展示方式（异形排版/沉浸交互）
🎬 收束        联系方式，一个清晰的下一步
```

要点：作品集本身就是最大的作品。展示方式比作品内容更能体现水平。

---

## 活动页 / 营销页（Campaign）

```
⚡⚡ 双重开场   极强冲击，撞色/大字/动效一起上，先声夺人
🔥 早高潮      活动核心信息（什么活动/什么好处）尽快给到，别让用户等
📈 推进        玩法/规则/亮点，节奏快，密度高
🫁 短呼吸      一个过渡，但别太长，活动页要保持热度
🎬 收束        强行动召唤 + 紧迫感（倒计时/名额/限时）
```

要点：活动页的节奏是"快"，热度不能掉。高潮可以提前，呼吸要短。

---

## 通用导演口诀

1. **开场定生死**：首屏不立住气质，后面全废。
2. **强弱交替**：连续两屏一样强 = 都不强；连续两屏一样弱 = 睡着了。
3. **高潮留给签名时刻**：把最敢的设计放在节奏高点，前面都是铺垫。
4. **收束要干脆**：观众该走的时候让他带着一个明确动作或一丝余味走，别拖。
5. **移动端重新导演**：竖屏是另一段节奏，不是横屏的压缩，该重排就重排。
